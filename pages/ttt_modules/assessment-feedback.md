## Overview

This module explores various assessment strategies and feedback techniques to evaluate learning outcomes and support learner development. Learn how to design assessments and provide constructive feedback.

## Learning Objectives

By the end of this module, participants will be able to:

- Design formative and summative assessments
- Provide constructive and actionable feedback
- Use assessment data to improve training
- Implement self and peer assessment strategies

## Key Topics

### Types of Assessment
- Formative vs. summative assessment
- Diagnostic assessment
- Self and peer assessment
- Authentic assessment in bioinformatics training

### Assessment Design
- Aligning assessments with learning objectives
- Creating rubrics and criteria
- Multiple choice, practical exercises, and projects
- Online assessment tools and platforms

### Feedback Strategies
- Principles of effective feedback
- Timing and delivery of feedback
- Constructive criticism and positive reinforcement
- Feedback for different learning contexts

### Evaluation and Improvement
- Collecting learner feedback
- Analyzing assessment results
- Using data to improve training
- Continuous improvement cycles

## Practical Activities

- Design an assessment for a training session
- Create a rubric for evaluating practical work
- Practice giving feedback in different scenarios
- Develop a feedback collection strategy

## Tools and Resources

- Assessment templates and examples
- Rubric generators and tools
- Feedback frameworks
- Survey and evaluation tools

## Assessment

Participants will demonstrate understanding through:
- Design of a complete assessment strategy
- Peer feedback practice
- Reflection on feedback received
- Development of an evaluation plan
